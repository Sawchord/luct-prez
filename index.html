luCT

Audit Certificate Transparency logs directly in Firefox

⭐ View on GitHub 📊 Presentation slides 💬 Join Matrix Chat

Why this exists

Certificate transparency is a protocol and ecosystem to make the Web more secure by requiring certificate authorities to publish their activity to publicly verifiable certificate transparency logs.

luCT will fetch and verify log entries for websites that you visit in the background.

Try it for yourself!

luCT is still pre-alpha and has not been published yet. As soon as an alpha release becomes available, it will be published here. In the meantime, you can download and test the current development version like this:
  1. Download the development version
  2. Unzip it. There should be luct.xpi file
  3. Type about:debugging into your browser
  4. Load the luct.xpi
❗CAUTION: The tool is still in very early development. Please treat it as a toy at this point in time. Do not rely on the output of the tool in any way!❗

Features and roadmap

Pre-alpha roadmap

  • ✅ Validate inclusion proofs of signed certificate timestamps
  • ✅ Fetch and update signed tree heads
  • ✅ Validate extension proofs
  • ✅ static-ct-api support
  • ✅ Oblivious TLS proxy to preserve privacy
  • 🚧 UI dashboard
  • 🚧 Publish in mozilla extension store

Alpha roadmap

  • 🚧 STH checkpoint verification
  • 🚧 Oblivious TLS proxy for CLI
  • 🚧 Root denylist with timelock

Post alpha roadmap

  • 🚧 DNS over HTTPs
  • 🚧 CAA and TLSA validation
